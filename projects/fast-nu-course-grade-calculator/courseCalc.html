By no means this is an official calculator nor it uses a formula formally approved by any of the authorities at FAST-NU. The formula for calculating the grade has been derived using citations from Rutgers University and Wikipedia.


You can read more about the formula here: Standard Score, but here's a brief overview.

C = 4.0 - ((M - A) / S)
Z = (U - A) / S
GPA = C + Z

C — Course Factor, M — Maximum Marks
A — Average Marks
Z — Z Score, U — User Marks
GPA — GPA Value
